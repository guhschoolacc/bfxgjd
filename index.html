Use this to assign roles. This calls a Cloud Function you must deploy (see comments below).
Quick tips
You can get a user's UID from Firebase Console → Authentication → Users. Custom claims are set server‑side (Cloud Function or Admin SDK).
Firestore rules (example) 
// Firestore rules snippet
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /global_chat/{msg} {
      allow read: if request.auth != null;
      allow create: if request.auth != null;
      allow delete, update: if request.auth.token.admin == true;
    }
  }
}
Deploy rules in Firebase Console → Firestore → Rules. Adjust for production.
